# Install the .NET SDK

The Box .NET SDK is available for both .NET Framework 4.5 and .NET Core 1.0 or
above. 

## .NET

Run the following command using Nuget.

```sh
PM> Install-Package Box.V2
```

## .NET Core

Run the following command using Nuget.

```sh
PM> Install-Package Box.V2.Core
```
